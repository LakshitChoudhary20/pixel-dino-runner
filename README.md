# Pixel Dino Runner

A Python + Pygame recreation of the classic Chrome offline Dinosaur Runner game, featuring smooth gameplay, spike variations, clouds, and polished mechanics.

---

## Features

- Enlarged dino for correct proportions
- Two obstacle types:
  - Single spike
  - Triple spike
- Cloud movement and parallax effect
- Smooth jumping physics
- Accurate ground collision
- Score system that increases over time
- Restart option after game over
- Simple, clean background

---

## Controls

| Key | Action |
|------|---------|
| SPACE | Jump |
| ENTER | Restart after game over |
| ESC / Close Window | Exit the game |

---

## Requirements

You need **Python 3.x** installed.

Install pygame using:

```bash
pip install pygame
```
